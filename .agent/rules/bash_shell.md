# Preferred Shell Configuration

The preferred shell for this workspace is Bash. All terminal commands should be executed using Git Bash (`C:\Program Files\Git\bin\bash.exe`) when running on Windows.

## Usage Guidelines:
1.  Prefer Bash-compatible commands over PowerShell.
2.  If running a command through the `run_command` tool, wrap it in `bash -c "...""` if necessary, or ensure the environment is set to Bash.
3.  The VS Code Workspace and User settings have been updated to use Git Bash as the default integrated terminal profile.
